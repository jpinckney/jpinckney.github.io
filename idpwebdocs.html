idp_web_docs

There is a certain passion i have about teaching others and helping to ease the difficulty with achieving the same level of success. This project brings to life my passion to help and teach others while they gain a greater understanding of code.

The idp_web_docs live site can be found here

idp_web_docs source code can be found here

This project was built with React as the framework, PostgreSQL for the database, Javacript for the logic and css for the styling.

The main page is designed for the user to pick a technology that they want to learn more about.

The Main content page will display information depending on what the user is requesting.
