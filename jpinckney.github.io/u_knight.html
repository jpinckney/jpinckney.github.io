U-Knight

U-Knight is a mix between a gamer social media platform and the ability to organize event / tournaments locally and online.

The U-Knight live site can be found here

U-Knight source code can be found here

This project was built with React as the framework, PostgreSQL for the database, Javacript for the logic and a mix between css and material UI for the styling.

The landing page is where you can immediately see highlights of a recent popular event as well as a list of upcoming events.

The search page is where you are able to filter through what events you will be interesting in such as online / local or Playstation / Xbox.

The host event page is where you can submit a picture to represent your event / tournament and fill out the info required that will draw people to your event.
